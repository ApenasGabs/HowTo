GitHelp / Instalação do GitHub

Github Desktop

    Vantagens

  • Interface gráfica amigável
  • Curva de aprendizado menor
  • Funciona sem a internet

    Desvantagens

  • Não tem todas as funcionalidades
  • Não está disponível em todas plataformas
  • Sem possibilidade de tradução

Github Web

    Vantagens

  1. Está disponível em todas plataformas
  2. Possui mais funções
  3. Possibilidade de tradução

    Desvantagens

  1. Interface gráfica não tão amigável
  2. Curva de aprendizado maior
  3. Não funciona sem a internet

Github Via Terminal

Vantagens

Agilidade:
Mais Agilidade para efetuar as funções do git;
Didático:
O uso do Github via terminal é Didático;
Funcionamento
Em Diversos SOs, Terminais e IDEs de diversas Linguagens

Desvantagens

Aprendizado:
Curva de aprendizado é bem maior e mais complexa
Acesso:
Precisa sempre de um terminal com teclado para ter acesso;
Funcionamento
Não Funciona 100%, sem conexão de Internet.
