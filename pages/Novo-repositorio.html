Para criar um novo repositório, após ter baixado e feito login, no canto superior esquerdo em “file” (arquivo) acesse a opção “new repository” (novo repositório)


Já no GitHub Web Para criar um novo repositório, após ter feito login, no canto superior esquerdo acesse a opção “new” (novo)


No campo “name ” coloque o nome do seu repositório, no “description” a descrição do seu projeto e no “local path” o lugar do seu computador onde seu projeto ira ficar armazenado após terminar de preencher os campos clique em “create repository”

Por fim, o último passo será publicar o repositório criado, para publicar basta clicar na opção “Publish repositoy”

Antes de finalizar, escolha se quer que o repositório seja público ou privado nessa sessão
