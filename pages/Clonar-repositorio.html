Clonando Respositório

A função de clonar um repositório no Github, consiste basicamente em baixar o repositório no seu desktop.

Ao abrir o GitHub Desktop escolha a segunda opção "clonar um repositório da internet "


Após escolher a opção clonar, uma tela para clonar o repositório irá aparecer, escolha a opção GitHub.com e selecione o repositório que deseja clonar, note que irá aparecer na barra Local Path
o local onde será armazenado o repositório dentro do seu Desktop, caso prefira você pode escolher outro local no botão Choose...

Quando estiver tudo certo basta clicar em clone e o repositório será baixado no seu Desktop
