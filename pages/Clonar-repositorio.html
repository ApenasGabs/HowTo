Clonando Respositório

A função de clonar um repositório no Github, consiste basicamente em baixar o repositório no seu desktop, você pode optar por clonar pelo GitHub Web ou pelo GitHub Desktop, a seguir mostrarei o passo a passo em ambos...

Desktop GitHub Web GitHub

Ao abrir o GitHub Desktop escolha a segunda opção "clonar um repositório da internet "

Quando abrimos o GitHub Web inicialmente nos é mostrado no canto esquerdo a lista de todos os nossos repositórios, para clonar você deve começar selecionando o repositório que deseja clonar

Após escolher a opção clonar, uma tela para clonar o repositório irá aparecer, escolha a opção GitHub.com e selecione o repositório que deseja clonar, note que irá aparecer na barra Local Path o local onde será armazenado o repositório dentro do seu Desktop, caso prefira você pode escolher outro local no botão Choose...

Em seguida a pagina será direcionada para o repositório, com algumas opções e informações sobre o mesmo, vá na opção em destaque code e clique para abrir as opções

Quando estiver tudo certo basta clicar em clone e o repositório será baixado no seu Desktop

Aparecerão algumas opções clique na opção Download ZIP e o repositório será baixado
